Etapas do Projeto

1. Controle Local Básico

Objetivo: Desenvolver sistema de controle simples sem integração de rede

Atividades:

  • Configurar CLP S7-1215 no TIA Portal
  • Programar lógicas básicas de automação
  • Criar interface local simples
  • Realizar testes funcionais
  • Documentar todas as configurações

Entregáveis:

  • Programa funcional do CLP
  • Documentação técnica completa
  • Relatório de testes realizados

2. Planejamento PROFINET

Objetivo: Estudar e planejar implementação da rede industrial

Atividades:

  • Definir topologia de rede adequada
  • Especificar endereçamento IP
  • Planejar integração de equipamentos
  • Elaborar cronograma detalhado
  • Validar arquitetura proposta

Entregáveis:

  • Diagrama de arquitetura de rede
  • Plano de endereçamento completo
  • Especificações técnicas

3. Demonstração PROFINET

Objetivo: Implementar rede funcional conectando CLPs e IHM

Atividades:

  • Configurar rede PROFINET no TIA Portal
  • Integrar S7-314C-2PN/DP e S7-1215
  • Conectar IHM KTP700 BASIC à rede
  • Programar comunicação entre dispositivos
  • Testar comunicação e diagnóstico

Entregáveis:

  • Sistema PROFINET operacional
  • Programas de teste validados
  • Manual de operação completo

4. Sistema Supervisório

Objetivo: Desenvolver interface de supervisão integrada

Atividades:

  • Configurar Elipse E3 para PROFINET
  • Desenvolver telas de supervisão
  • Implementar alarmes e históricos
  • Integrar Node-RED para análises
  • Criar dashboard web responsivo

Entregáveis:

  • Sistema supervisório funcional
  • Dashboard web interativo
  • Manual completo do usuário

5. Integração Avançada

Objetivo: Expandir sistema com sensores e controles inteligentes

Atividades:

  • Integrar sensores indutivos e capacitivos
  • Configurar comunicação HART
  • Implementar controle de processo
  • Desenvolver algoritmos avançados
  • Validar performance integrada

Entregáveis:

  • Sistema de controle expandido
  • Algoritmos implementados
  • Relatório de performance
